　　再者，她也不认为他真需要她救。

　　她呼吸紊乱，手按在谢无极的腰间，缓缓凝结力量。
　　她要试试他到底是不是真的晕过去了。
　　他身上本来就全都是窟窿，她这一掌下去，他就算死不了，短时间内也无法再耀武扬威。
　　如果他是装的，肯定不会坐以待毙。

　　她急忙站起来，朝留下的最后一名守卫伸出手。

　　黎瑶浑身一震。
　　他们想把她丢下？
　　因为她刚才对谢无极出手？
